cd ----------------------------------------------

- note: cd can not be executed in the child process since the changes will
    not be made in the main program

- absolute path:
    - /home/user/documents
    - /var/log
    - /usr/local/bin
    - /
- relative path
    - documents/
    - ./documents
    - ../logs
    - ../../
- updates env: PWD and OLDPWD
